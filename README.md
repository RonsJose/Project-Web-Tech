# Web Technology Project (Game Store Website)
A multi-page website designed to advertise and manage purchases for popular games such as Minecraft, Rainbow Six Siege, and ARK.

The project includes interactive pages, a checkout system, and game-specific features, demonstrating frontend development and basic client-side logic.

# Features:
* Dedicated landing pages for each game:
  * Minecraft
  * Rainbow Six Siege
  * ARK
* Display game information, visuals, and promotional content
* Checkout Manager
  *  Users can select multiple games
  *  Dynamically calculates total cost based on selection
  *  Acts as a basic shopping cart system
  *  Handles user interaction and selection logic

# Minecraft Page (Dynamic Features)
* Supports different Minecraft versions
* Allows interaction based on selected version
* Displays version-specific information/features
